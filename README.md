# MenuInteractivo

Proyecto académico orientado al desarrollo de un sistema de menús interactivos para museos, tomando como caso inicial el Museo de Malvinas de San Carlos de Bariloche.

## Integrantes

- Mara Fillol
- Sofía Silva
- Dana Mancilla

## Objetivo

Desarrollar una plataforma que permita administrar y visualizar contenido histórico, educativo y multimedia mediante una interfaz interactiva para los visitantes y un sistema de gestión para administradores.

## Tecnologías utilizadas

- Node.js
- Express
- SQLite
- Git
- GitHub

## Estado actual

Actualmente se encuentra implementada una primera versión del backend, incluyendo:

- Base de datos SQLite.
- Gestión básica de menús.
- Gestión básica de fichas.
- API REST en desarrollo.

## Próximos pasos

- Desarrollo del panel administrativo.
- Gestión de contenido multimedia.
- Implementación de etiquetas y relaciones.
- Desarrollo de la interfaz interactiva para visitantes.
